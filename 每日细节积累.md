## getchar()吸收回车

```
#include <stdio.h>
int main()
{
    char c;

    while ((c = getchar()) != 'a')
        printf("%c",c);
    getchar();
    c=getchar();

    printf("%c",c);

    return 0;
}
```
**输入ssssa回车**  
**输出ssss**  
**如果注释掉13行的`getchar()；`**  
**就输出ssss回车**

----
`putchar(10);`是输出10对应的ASCII表中的字符-换行
